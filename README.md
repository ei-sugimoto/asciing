# asciing

## about
asciing is CLI to create asCII art. If you put str, these automatically will converse ascII art.

```
 | (_| | \__ \ | (__  | | | | | | | | | (_| |
  \__,_| |___/  \___| |_| |_| |_| |_|  \__, |
                                       |___/

```

## install 
```bash
go install "github.com/ei-sugimoto/asciing
```

## usage
```bash
asciing create hello
```
